Tunnels
=======

This tunnels https to http.

When you are using pow to develop rack app,
`https://yourapp.dev` will be tunneled to `http://yourapp.dev`.

![image](http://i.imgur.com/5F9tJ.png)

Installation
------------

    $ gem install tunnels

Run
---

    $ tunnels

or

    $ sudo tunnels
