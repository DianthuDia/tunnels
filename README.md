Tunnels
=======

![image](http://i.imgur.com/5F9tJ.png)

This tunnels https to http.

You can run the [Pow](http://pow.cx/) over SSL!

Installation
------------

    $ gem install tunnels

Run
---

    $ tunnels

or

    $ sudo tunnels
