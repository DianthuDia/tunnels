Tunnels
=======

![image](http://i.imgur.com/5F9tJ.png)

Tunnels is a proxy to http from https.

You can run the [Pow](http://pow.cx/) over SSL!

Installation
------------

    $ gem install tunnels

Run
---

    $ tunnels

or

    $ sudo tunnels
